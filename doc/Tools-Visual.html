Development Tools

Online Bookstore  |  Tools & Technologies Used During Development  |  February 2026

This document lists all the tools, frameworks, libraries, and software used during the development, testing, and deployment of the Online Bookstore application. They are organized by category: Backend, Frontend, Database, Build Tools, Development Environment, Version Control & Deployment, and Design & Documentation.

1. Backend Tools

P
PHP
Version 8.4.14
The core server-side programming language used to build the entire application logic, controllers, models, middleware, and service providers.
https://www.php.net
L
Laravel Framework
Version 12.38.1
The PHP web application framework providing MVC architecture, Eloquent ORM, Blade templating, routing, middleware, authentication, and artisan CLI commands.
https://laravel.com
B
Laravel Breeze
Version 2.3
A lightweight authentication scaffolding for Laravel providing login, registration, password reset, email verification, and profile management out of the box.
https://github.com/laravel/breeze
C
Composer
Version 2.x
PHP dependency manager used to install and manage all backend packages including Laravel, Breeze, and other PHP libraries defined in composer.json.
https://getcomposer.org
A
Artisan CLI
Built into Laravel
Laravel's command-line interface used for running migrations, seeding the database, generating controllers/models, clearing caches, and starting the development server (php artisan serve).
https://laravel.com/docs/artisan

2. Frontend Tools

B
Blade Templating Engine
Built into Laravel
Laravel's built-in templating engine used to create all views (.blade.php files) with template inheritance, components, directives, and conditional rendering.
https://laravel.com/docs/blade
T
Tailwind CSS
Version 3.x
A utility-first CSS framework used for styling all pages — providing responsive design, flexbox/grid layouts, spacing, typography, and color utilities without writing custom CSS.
https://tailwindcss.com
A
Alpine.js
Version 3.x
A lightweight JavaScript framework used for interactive UI behaviors — dropdown menus, modal dialogs, form toggles, and reactive components directly in Blade templates.
https://alpinejs.dev
Ax
Axios
Version 1.x
A promise-based HTTP client used for making AJAX requests from the frontend, included as a default dependency in Laravel's JavaScript bootstrap file.
https://axios-http.com

3. Database Tools

M
MySQL
Version 8.4.3
The relational database management system (RDBMS) storing all application data — 13 tables including users, books, cart_items, orders, order_items, and Laravel infrastructure tables.
https://www.mysql.com
P
phpMyAdmin
Web-based
A web-based database administration tool used to manage the MySQL database — browse tables, run SQL queries, import/export data, and manage users and permissions.
https://www.phpmyadmin.net
E
Eloquent ORM
Built into Laravel
Laravel's Active Record ORM used for database interactions — defining models (User, Book, CartItem, Order, OrderItem), relationships (hasMany, belongsTo), and query building.
https://laravel.com/docs/eloquent
Mi
Laravel Migrations
Built into Laravel
Database version control system used to define and modify the database schema programmatically — creating tables, adding columns, defining foreign keys, and setting indexes.
https://laravel.com/docs/migrations

4. Build & Package Tools

V
Vite
Version 7.2.2
A modern frontend build tool used for compiling and bundling CSS (Tailwind) and JavaScript (Alpine.js, Axios) assets with hot module replacement (HMR) during development and optimized production builds.
https://vitejs.dev
N
Node.js
Version 22.x
JavaScript runtime environment required to run Vite, npm, and all frontend build tooling. Provides the execution environment for compiling assets.
https://nodejs.org
npm
npm
Version 10.x
Node Package Manager used to install and manage all frontend dependencies defined in package.json — including Vite, Tailwind CSS, Alpine.js, Axios, and PostCSS.
https://www.npmjs.com
Pc
PostCSS
Version 8.x
A CSS transformation tool used as part of the Tailwind CSS build pipeline — processing utility classes, autoprefixing, and optimizing the final CSS output.
https://postcss.org

5. Development Environment

La
Laragon
Full Edition
A portable, isolated local development environment for Windows providing Apache, MySQL 8.4.3, PHP, and phpMyAdmin in a single package. Used as the primary development server stack.
https://laragon.org
VS
Visual Studio Code / Windsurf
Latest
The primary code editor / IDE used for writing all PHP, Blade, JavaScript, CSS, and configuration files. Provides syntax highlighting, IntelliSense, integrated terminal, and extensions for Laravel development.
https://code.visualstudio.com
T
Windows Terminal / PowerShell
Windows 10/11
The command-line interface used for running Artisan commands, Composer, npm, Git, and MySQL CLI operations throughout the development process.
https://github.com/microsoft/terminal
FF
Web Browser (Firefox / Chrome)
Latest
Used for testing the application UI, debugging with Developer Tools (F12), inspecting network requests, viewing console logs, and testing responsive design across breakpoints.
https://www.mozilla.org/firefox

6. Version Control & Deployment

G
Git
Version 2.x
Distributed version control system used to track all code changes, manage branches, create commits with conventional commit messages, and synchronize with the remote repository.
https://git-scm.com
GH
GitHub
github.com
Cloud-based Git hosting platform used as the remote repository for the project. All code, documentation, and diagram files are pushed to github.com/imran-Haram/online-bookstore.
https://github.com/imran-Haram/online-bookstore

7. Design & Documentation Tools

SVG
SVG (Scalable Vector Graphics)
W3C Standard
Used to create all visual diagrams — DFD (Data Flow Diagrams), ERD (Entity Relationship Diagram), Use Case Diagram, and Database Design Diagram — as resolution-independent vector graphics embedded in HTML.
https://www.w3.org/Graphics/SVG/
Md
Markdown
CommonMark
Lightweight markup language used for writing the main project documentation (ProjectDocumentation.md) with structured headings, tables, code blocks, and formatted text.
https://commonmark.org
H
HTML5 / CSS3
W3C Standard
Used to create all visual documentation files (DFD-Visual.html, ERD-Visual.html, UseCase-Visual.html, DatabaseDesign-Visual.html, DatabaseQueries-Visual.html) with modern styling and print-ready layouts.
https://developer.mozilla.org
