Use Case Diagram

Actors

ActorTypeDescription
Customer Primary A registered user who can browse, search and filter books, manage a shopping cart, place orders, view order history, and manage their profile. Must register and login to perform cart/order/profile operations.
Administrator Primary A privileged user (is_admin = true) who can access the admin dashboard, manage books (CRUD), manage orders (view and update status), and view registered users. Shares login/logout with the Customer actor.

Use Case Descriptions

Customer Use Cases

IDUse CaseActorDescriptionController / Route
UC-01RegisterCustomerCreate a new account by providing name, email, and passwordRegisteredUserController → POST /register
UC-02LoginCustomer, AdminAuthenticate with email and password to access the systemAuthenticatedSessionController → POST /login
UC-03LogoutCustomer, AdminTerminate the current session and redirect to the homepageAuthenticatedSessionController → POST /logout
UC-04Browse BooksCustomer, AdminView the paginated book catalog with title, author, price, and stockBookController@index → GET /books
UC-05View Book DetailsCustomerView full details of a specific book including description and imageBookController@show → GET /books/{book}
UC-06Search / Filter BooksCustomerSearch by keyword or filter by category to find specific booksBookController@index → GET /books?search=&category=
UC-07Add to CartCustomerAdd a book to the shopping cart with a specified quantityCartController@store → POST /cart
UC-08Update / Remove CartCustomerChange item quantity or remove items from the cartCartController@update|destroy → PATCH|DELETE /cart/{id}
UC-09Checkout & Place OrderCustomerReview cart, confirm checkout, and create an order with all cart itemsOrderController@create|store → GET|POST /checkout
UC-10View Order HistoryCustomerView a list of all past orders with status, total, and dateOrderController@index|show → GET /orders
UC-11Edit ProfileCustomerUpdate name, email, or change passwordProfileController@update → PATCH /profile
UC-12Delete AccountCustomerPermanently delete account and all associated dataProfileController@destroy → DELETE /profile
